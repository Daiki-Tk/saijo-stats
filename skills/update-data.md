# データ更新スキル

## このスキルを使う場面
新年度のデータをCSVに追加・更新するとき

## 手順
1. data/フォルダの対象CSVをExcelで開く
2. 最終行に「年度,項目名,値」の形式で新しいデータを追加する
3. 保存後にダッシュボードをブラウザで開いてグラフを確認する
4. 表示に問題がなければGitHubにプッシュする

## CSVのフォーマット確認
- 1行目はヘッダー（年度,項目名,値）
- 年度は西暦4桁（例：2027）
- 値は数値のみ（単位・カンマなし）
- 文字コードはUTF-8で保存すること

## よくあるミス
- カンマが全角になっている
- 値に「,」（千区切り）が入っている
- 文字コードがShift-JISになっている
